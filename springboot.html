spring family

常说的Spring，指的是Spring Framework。 Spring Boot 可以以一种更便捷的方式使用Spring各组件，比如xxx-starter

spring ability

  • 微服务
  • 响应式编程（非阻塞，异步数据流，减少资源需求）
  • 分布式
  • web
  • 无服务： FaaS， 函数即服务
  • 事件驱动
  • 批处理

SpringBoot

Spring技术栈的一站式开发框架，是简化Spring技术栈的快速开发脚手架 SpringBoot 2.0 （基于java8实现）

  • 配置
  • web
  • junit5
  • 和其他组件的整合

advantage： ​ 简单、快捷，但是完备

disadvantage： ​ 版本迭代太快 ​ 封装太深，内部远离复杂（有成本，需要精通 Spring）
