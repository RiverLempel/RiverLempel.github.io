basic

第一版发布于1983年。

  • 对异构平台的支持较好
  • 对BI的支持
  • 数据库优化技术（性能不错）
  • 可靠性
  • 开放性/可移植性（支持最新的国际标准）
  • 安全性
  • 可处理海量数据

concepts

  • instance
  • database
  • table space DMS (database management space) : one or more containers SMS (system management space): one directory
  • container 指定了数据的物理存储位置 three types: File, Directory, Device
