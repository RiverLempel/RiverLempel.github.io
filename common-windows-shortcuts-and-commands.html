common windows shortcuts and commands

 Mon 05 April 2021   In 编程   :)

shortcut

​    Win + R， cmd : 命令行

​    Win + E：打开我的电脑

​    Win + D： 最小化和恢复

​    Ctrl+N： 复制当前窗口

​    Alt+F4 or Ctrl + W: 关闭窗口

​    Ctrl+Z，撤销操作

​    Ctrl+Y，还原最近的撤销


​    Ctrl + Shift + N : create new directory


​    Ctrl + D: move to 回收箱

​    Delete：删除

​    Shift + Delete: 永久删除

​    

​    Alt + Tab： 切换窗口

​    Win + T : 切换窗口预览 （好用）

​    长按 Alt ： 快捷操作提示 （好用）

​    Win  + number： 切换任务栏的任务

file operation part1

dir 
/a
 d: directory
 a: files
/b : bare format, only name
/l : use lower case (其实不区分大小写）
/o 
 n: sort by name
 s: sort by size
 d: sort by time (asc)
 -: 反

example：
// 列出所有文件
dir /aa/b
// 列出所有文件夹，使用小写名称
dir /ad/b/l 

// 按大小排序列出文件名
dir /aa/b/os

file operation part2

copy
del
move

tree: show file and directory with tree
